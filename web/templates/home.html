Lurker Bot

What is this?

Raptr and Xfire went out, so this bot fills the void. It looks at your status in Discord and keeps track of how long you play games!

How to start tracking

1
Have access to the bot
Only one of the following 2 options must be true
a
Join the Lurker Server
b
Invite Lurker to your Server
2
On any server where Lurker Bot is present, message the bot "addme".
To stop tracking, message the bot "removeme".
3
The bot will respond to your message no matter what. Verify you receive a message.
4
You're done!
NOTE: Be sure to toggle "Display currently running game as a status message" on.

What am I tracking?

Your status / Presence on Discord. If it changes and you're being tracked, I parse that information to determine what game you are playing as well as the game state and information available to me via presence.

Only the bot is tracking items, and the code is readily available on Github
